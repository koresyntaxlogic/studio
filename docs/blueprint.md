# **App Name**: KoreSyntax Logic

## Core Features:

- Interactive UI: Modern aesthetic UI components built with React.
- Service Showcase: Interactive components that showcase services.
- Product Catalog: Components that present a structured catalog of products.
- Contact Form: Collect user data using an interactive and user-friendly interface.
- Lead Capture AI: Leverage a generative AI tool that collects user-provided project details from the contact form and suggests an appropriate service tier. 

## Style Guidelines:

- Primary color: Cyan neon (#00F3FF) for highlights and interactive elements, providing a modern, digital feel.
- Background color: Dark navy (#0A101A) provides a futuristic look.
- Accent color: Purple neon (#7000FF) for secondary highlights and calls to action, complementing the primary color with a vibrant contrast.
- Headline font: 'JetBrains Mono' monospace font to reflect computer science
- Body font: 'Inter' sans-serif font to provide clarity.
- FontAwesome icons to enhance the UI.
- Subtle animations to improve the user experience
